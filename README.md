# Rock Paper Scissors

A simple Rock-Paper-Scissors game built with HTML, CSS, and JavaScript.

## How to Play
- Click one of the buttons (Rock, Paper, or Scissors) to make your choice.
- The computer will randomly select its choice.
- The game keeps score and ends when someone reaches 5 points.
- Click "Play again" to restart.

## Preview
[Play the game here](https://yourusername.github.io/your-repo-name/)

## Features
- Button-based UI (no typing required)
- Live score updates
- Round-by-round results
- Reset functionality

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## Setup
1. Clone the repository.
2. Open `index.html` in your browser, or serve it with a local server.
